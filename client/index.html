FydeDrop // 燧炻传送

Transfer files across devices in FydeOS // FydeOS 中的跨设备文件传输
FydeDrop lets you transfer files between P2P connection. It supposed that your devices have the same public IP address and are not using a proxy, network tunnel or VPN. // “燧炻传送” 可让您通过 P2P 连接传输文件。使用条件是您的设备处于同一公网 IP 地址，且没有使用代理、隧道或 VPN 等程序。
